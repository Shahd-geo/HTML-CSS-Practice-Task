# Deiterhaus Landing Page

A responsive landing page inspired by the Deiterhaus website design, built using HTML5 and CSS3.

## Features

- Responsive Design
- Flexbox Layout
- Navigation Bar
- Hero Section
- Call-to-Action Button
- Background Image
- Media Queries
- Responsive Typography using `clamp()`

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
